# 互联网营销师

文档记录了互联网营销师学习的基础知识，内容包含**运营**、**拍摄**、**剪辑**、**直播**、**文案**等内容,涉及**抖音**、**小红书**等平台。

在此，特别感谢 况老师 的辛苦付出，为大家梳理文档。

文档将持续更新相关内容，并在 [GitHub](https://github.com) 开源，感兴趣的朋友可参与添加、修改、校验。
